## 시스템 개요

### 이 시스템은 무엇인가요?
**조특법 세액공제 자동 판정 시스템**은 사업자가 받을 수 있는 **세액공제**를 자동으로 찾아주는 서비스입니다.

### 세액공제란?
- **세금을 줄여주는 혜택**입니다
- 예: 직원을 채용하면 → 1인당 최대 1,200만원 세금 감면
- 예: 시설 투자를 하면 → 투자액의 10% 세금 감면

### 누가 사용하나요?
- ✅ **중소기업 대표님**
- ✅ **세무사/회계사**
- ✅ **경영컨설턴트** (우서기님 같은 전문가)

## 규칙 조회 사용법

### 📚 규칙 조회 페이지로 이동

1. 메인 페이지에서 **"규칙 조회"** 버튼 클릭
2. 20개 세액공제 규칙이 카테고리별로 나타납니다

### 카테고리 설명

#### 1️⃣ **고용 관련** (5개)
```
👥 직원을 고용하면 받을 수 있는 혜택
- 고용증대 세액공제: 직원 늘리면 1인당 최대 1,200만원
- 청년 고용: 청년(15-34세) 채용하면 추가 1,200만원
- 장애인 고용: 장애인 채용하면 1인당 960만원
- 경력단절여성: 재고용 시 1인당 1,100만원
- 사회보험료: 보험료의 25% 공제
```

#### 2️⃣ **중소기업 관련** (5개)
```
🏢 중소기업이라면 받을 수 있는 혜택
- 중소기업 특별세액감면: 세금의 5~10% 감면
- 창업 감면: 창업 후 5년간 50~100% 감면
- 지방 이전: 수도권→지방 이전 시 7년간 100% 감면
- 사회적기업: 인증받으면 50~100% 감면
- 청년창업: 34세 이하 창업 시 100% 감면 (연 2억 한도)
```

#### 3️⃣ **투자·시설** (4개)
```
🏭 시설이나 장비를 구매하면 받을 수 있는 혜택
- 생산성향상시설: 자동화 설비 등 투자액의 3~10%
- 에너지·환경시설: 친환경 설비 투자액의 3~10%
- 안전시설: 화재예방 등 투자액의 3~10%
- 스마트공장: 자동화 설비 투자액의 5~15%
```

#### 4️⃣ **연구개발** (3개)
```
🔬 연구개발을 하면 받을 수 있는 혜택
- 연구개발비: R&D 비용의 5~30% 공제
- 디자인 개발: 디자인 비용의 5~25% 공제
- 신기술(AI/IoT): 신기술 개발비의 10~30% 공제
```

#### 5️⃣ **기타** (2개)
```
📋 기타 항목
- 기부금: 기부금액의 15% 공제
- 업무용 차량: 차량 비용 한도 확인
```

### 규칙 상세보기 방법

1. 각 규칙 항목을 **클릭**하면 펼쳐집니다
2. **적용 요건**: 어떤 조건을 만족해야 하는지
3. **공제 금액/율**: 얼마나 받을 수 있는지

**예시**: "고용증대 세액공제" 클릭 시
```
✅ 적용 요건
  • 상시근로자 증가 인원: 최소 1명
  • 기업 규모: 중소기업, 중견기업, 대기업

💰 공제 금액/율
  • 중소기업: 1인당 1,100만원 (수도권 외 1,200만원)
  • 중견기업: 1인당 900만원 (수도권 외 1,000만원)
  • 대기업: 1인당 450만원 (수도권 외 500만원)
```

## 세액공제 판정 신청 방법

### 🚀 5단계로 쉽게 신청하기

메인 페이지에서 **"새로운 판정"** 버튼을 클릭하면 시작됩니다.

---

### 📋 1단계: 사업자 기본 정보 입력

#### 입력 항목

| 항목 | 설명 | 예시 |
|------|------|------|
| **사업자등록번호** | 사업자등록증에 있는 번호 | `123-45-67890` |
| **회사명** | 정식 회사 이름 | `(주)테스트기업` |
| **대표자명** | 대표이사 이름 | `홍길동` |
| **기업 규모** | 선택: 중소/중견/대기업 | `중소기업` |
| **업종** | 사업 분야 | `제조업` |
| **소재지** | 회사 주소 | `서울특별시 강남구` |
| **수도권 여부** | 서울/경기/인천이면 수도권 | `수도권` 또는 `지방` |
| **과세연도** | 신고할 연도 | `2024` |

#### 💡 꿀팁!
- 사업자등록번호는 **하이픈(-) 포함**하여 입력하세요
- 기업 규모가 헷갈리면:
  - **매출액 400억원 이하** → 중소기업
  - **매출액 1,500억원 이하** → 중견기업
  - **그 이상** → 대기업

#### ✅ 완료 후
**"다음 단계"** 버튼을 클릭하면 자동으로 사업자 정보가 저장됩니다.

---

### 👥 2단계: 고용 관련 정보 입력

#### "고용 공제를 받지 않으시나요?"
→ **모두 0으로 두고 "다음 단계" 클릭하세요!**

#### 고용 공제를 받으시려면:

| 항목 | 언제 입력하나요? | 예시 |
|------|------------------|------|
| **총 상시근로자 수** | 현재 직원 수 | `15명` |
| **전년 대비 증가 인원** | 작년보다 늘어난 직원 수 | 작년 10명 → 올해 15명 = `5명` |
| **청년 정규직 증가** | 15-34세 청년 채용 수 | `3명` |
| **장애인 근로자** | 장애인 직원 수 | `1명` |
| **경력단절여성** | 3년 이상 경력단절 여성 재고용 | `2명` |
| **연간 총 급여액** | 직원 전체 급여 합계 | `500000000` (5억원) |
| **사회보험료** | 사업주가 낸 보험료 | `50000000` (5천만원) |

#### 💡 예시 시나리오
```
우리 회사는 작년에 직원 10명 → 올해 15명으로 늘었어요.
이 중에서 청년(20대) 3명이 포함됩니다.

입력:
✏️ 총 상시근로자 수: 15
✏️ 전년 대비 증가: 5
✏️ 청년 정규직 증가: 3
```

**결과 예상**:
- 고용증대 공제: 5명 × 1,100만원 = **5,500만원**
- 청년 추가 공제: 3명 × 1,200만원 = **3,600만원**
- **총 9,100만원 세금 절감!** 🎉

---

### 🏭 3단계: 투자 및 시설 정보 입력

#### "투자한 시설이 없으시나요?"
→ **"다음 단계" 클릭하세요!**

#### 투자한 시설이 있으시다면:

1. **"투자 항목 추가"** 버튼 클릭
2. 다음 정보 입력:

| 항목 | 선택/입력 | 예시 |
|------|-----------|------|
| **시설 종류** | 드롭다운에서 선택 | `자동화설비` |
| **투자 금액** | 실제 지출한 금액 | `50000000` (5천만원) |

#### 시설 종류 설명

```
🤖 자동화설비: 로봇, 자동화 기계 등
💻 정보시스템: ERP, MES 같은 시스템
♻️ 에너지절약시설: LED 조명, 태양광 패널 등
🌳 환경보전시설: 폐수처리, 대기오염 방지 시설
🚨 화재예방설비: 소화기, 스프링클러 등
🏭 스마트공장: 스마트공장 인증받은 설비
```

#### 💡 여러 개 투자했나요?
**"투자 항목 추가"** 버튼을 **여러 번** 클릭하여 모두 입력하세요!

**예시**:
```
투자 1: 자동화설비 3천만원
투자 2: 에너지절약시설 2천만원
→ 총 5천만원 투자 → 10% 공제 → 500만원 세금 절감!
```

#### "연구개발비가 없으시나요?"

## 자주 묻는 질문 (FAQ)

### Q1. 입력한 정보는 저장되나요?
**A**: 네! 사업자등록번호를 입력하면 자동으로 데이터베이스에 저장됩니다.

### Q2. 중간에 끊어도 되나요?
**A**: 1단계(사업자 정보)만 완료하면 나중에 이어서 할 수 있습니다.

### Q3. 잘못 입력했어요!
**A**: "이전" 버튼을 눌러서 이전 단계로 돌아가서 수정할 수 있습니다.

### Q4. 해당 항목이 없으면 어떻게 하나요?
**A**: 비워두거나 0으로 입력하고 "다음 단계"로 넘어가세요.

### Q5. 공제액이 0원으로 나와요!
**A**: 다음을 확인하세요:
- 증가 인원이나 투자 금액이 최소 기준을 만족하는지
- 기업 규모가 정확한지 (중소기업만 해당되는 공제가 많습니다)

### Q6. 계산 결과가 정확한가요?
**A**: 조세특례제한법을 기반으로 계산하지만, **참고용**입니다. 실제 신고 시 세무사와 상담하시기 바랍니다.

### Q7. 여러 회사를 동시에 관리할 수 있나요?
**A**: 네! 사업자등록번호가 다르면 각각 따로 관리됩니다.

### Q8. 모바일에서도 되나요?
**A**: 네! 반응형으로 제작되어 모바일에서도 사용 가능합니다.
