## 📖 1단계: 시스템 이해하기

### ❓ 세액공제가 뭔가요?

**쉽게 말하면**: 정부에서 주는 **세금 할인 쿠폰**입니다!

```
예시:
회사가 내야 할 세금: 1억원
세액공제 (할인): 2천만원
───────────────────────────
실제로 낼 세금: 8천만원 ✨

→ 2천만원 절약! 💰
```

### 💡 언제 받을 수 있나요?

다음 중 **하나라도** 해당하면 받을 수 있어요:

1. **직원을 새로 뽑았어요** → 1명당 최대 1,200만원!
2. **기계나 설비를 샀어요** → 구매액의 10%!
3. **연구개발을 했어요** → 비용의 30%!
4. **창업한 지 5년 안 됐어요** → 세금의 50~100%!

### 🎯 이 시스템이 해주는 일

```
[입력] 우리 회사 정보 (직원 수, 투자 금액 등)
    ↓
[처리] 자동으로 20개 항목 검토
    ↓
[출력] 받을 수 있는 혜택과 금액
```

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 입력하다가 창을 닫았어요!

**A**: 걱정 마세요!
- 1단계(사업자 정보)만 완료했으면 저장되어 있습니다
- 다시 접속해서 이어서 하시면 됩니다

### Q2. 잘못 입력했어요!

**A**: 두 가지 방법:
1. **"← 이전" 버튼**으로 이전 단계로 돌아가기
2. **처음부터 다시 시작하기** (새로운 사업자번호 입력)

### Q3. 0원 또는 입력 안 해도 되나요?

**A**: 네, 괜찮아요!
- 해당사항 없으면 **비워두거나 0** 입력하세요
- **"다음 단계"** 버튼 눌러서 넘어가면 됩니다

### Q4. 숫자를 어떻게 입력하나요?

**A**: 
```
✅ 올바른 입력:
  5천만원 → 50000000
  1억원 → 100000000

❌ 잘못된 입력:
  50,000,000 (쉼표 X)
  5천만원 (한글 X)
  50000000원 (단위 X)
```

### Q5. 공제액이 이상해요!

**A**: 다음을 확인하세요:
1. **기업 규모**가 정확한가요? (중소/중견/대기업)
2. **수도권 여부**가 정확한가요? (수도권/지방)
3. **증가 인원**이 맞나요? (작년과 비교)

### Q6. 결과를 저장하고 싶어요!

**A**: 
```
방법 1: [🖨️ 결과 인쇄] → PDF로 저장
방법 2: 화면 캡처 (Print Screen)
방법 3: 브라우저에서 우클릭 → "다른 이름으로 저장"
```

### Q7. 여러 회사를 동시에 관리할 수 있나요?

**A**: 네!
- 각 회사의 **사업자번호가 다르면** 각각 저장됩니다
- 회사마다 새로 판정하면 됩니다

### Q8. 모바일에서도 되나요?

**A**: 네, 됩니다!
- 스마트폰 브라우저로 접속하세요
- 화면이 작아서 PC가 더 편해요

### Q9. 실제로 이만큼 받을 수 있나요?

**A**: 참고용입니다!
- 이 시스템은 **예상 금액**입니다
- **실제 신고 시 세무사와 상담**하세요
- 법령 개정으로 금액이 달라질 수 있어요

### Q10. 에러가 나요!

**A**: 
```
1. 페이지 새로고침 (F5)
2. 브라우저 캐시 삭제
3. 다른 브라우저로 시도
4. 인터넷 연결 확인
```
